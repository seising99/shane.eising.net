# Eising.net

A deliberately small static personal site built with Astro.

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Notes

- The site is static: no framework islands, no client-side JavaScript, and no Tailwind.
- Shared HTML lives in `src/layouts/Layout.astro`.
- Route files live in `src/pages/`.
